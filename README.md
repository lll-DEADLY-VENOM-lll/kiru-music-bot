# YouTube URL API
A simple Node.js API to fetch YouTube video download links using yt-dlp.

## How to Install
1. Clone this repo: `git clone https://github.com/YOUR_USERNAME/repo-name.git`
2. Install dependencies: `npm install`
3. Start server: `node index.js`

## API Usage
`GET /api/download?url=YOUR_YOUTUBE_URL`
